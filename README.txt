# CSharp school assignment

### Description
(G) Create a console application that is a addressbook where you can add, remove, see all and see a single contact.
(VG) Create a MAUI/WPF application that is a addressbook where you can add, remove, update, see all and see a single contact.

### Execution
I created a class library where I have all my public Models and Services that l use in both my Console application and my MAUI application.
Console application (AddressBook) it's possible to add, remove, see all and see a single contact.
MAUI application (MauiAddressBook) it's possible to add, remove, update, see all and see a single contact.

I also added tests for both my applications.
